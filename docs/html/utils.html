xcell.mappers.utils.get_DIR_Nz(cat_spec, cat_photo, bands, zflag, zrange, nz, nearest_neighbors=10, njk=100, bands_photo=None)[source]

Implementation of the DIR algorithm that calibrates a photometric galaxy redshift distribution given a reference spectroscopic sample. The code makes use of sklearn nearest neighbors algorthim

Parameters

  • cat_spec (-) – catalog of spectroscopic samples.

  • cat_photo (-) – catalog of photometric samples.

  • bands (-) – bands for spectroscopic and photometric samples.

  • zflag (-) – name of the redshift field in the samples catalog.

  • zrange (-) – redshift range for the calibrated galaxy redshift distribution.

  • nz (-) – number of bins for calibrated galaxy redshift distribution.

  • 10 (- nearest_neighbors =) – number of nearest neighbors found by the algorithm.

  • njk (-) – Loop over JK region.

  • None (- bands_photo =) – bands of phometric catalog.

Returns

position of bins in redshift - dndz (Array): elements per bin - dndz_jk (Array):

Return type

  • zz (Array)
